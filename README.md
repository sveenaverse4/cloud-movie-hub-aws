# ☁️ AWS Cloud Movie Hub

This project demonstrates **Infrastructure as Code (IaC)** using Terraform to deploy a custom-styled web application on AWS.

### 🛠️ Tech Stack
* **Cloud:** Amazon Web Services (AWS)
* **Automation:** Terraform
* **Server:** Apache (httpd) on EC2
* **Frontend:** HTML5 / CSS3

### 💡 Project Goal
I built this to prove that cloud infrastructure can be automated, secure, and beautiful. It serves as a foundation for my journey into Cloud Engineering.
